Kirjutan siia veel ühe rea juurde!
git status  --näitab ära kõik kas on muutusied jne
git diff  --näitab erinevust failis mis on muudetud
git commit --commitib faili ära
git commit -m "Some Text"  --addib commiti 
git diff --staged  --näed muudatusi ka siis kui oled updateinud faili
git checkout nimi --läheb sinna branchi kuhu soovid
git branch nimi --tekitab selle branchi mida soovid
git checkout -b nimi --tekitab uue branchi ja läheb kohe sinna
git merge brenchname -- sulatab kokku või viib üheks selle brenchi kus oled sellega mida soovid sinna addida või kokku liita
